# Lâmpada Liga/Desliga. :bulb:

- Clic em "Liga" para ligar a lâmpada.
- Clic em "Desliga" para deslidar a lâmpada.
- Mantenha o mouse ensima da lâmpada para liga-la.
- Retire o mouse de cima da lâmpada para desliga-la.
